# Student Information Management System

## Degree Program
Computer Science

## Group Number
Group 7

## Project Overview
A web-based student information management system designed for primary and secondary schools in Tanzania. This system streamlines student record management by allowing school administrators to register students, view all records, and search for specific students using their registration number.

## Features
- Register new students with complete details
- View all student records in a sortable table
- Search for students by registration number
- Delete student records (with confirmation)
- Dashboard with student statistics and class distribution
